#### 礼物映射表
获取方式：
```$xslt
## 公共礼物映射
https://webconf.douyucdn.cn/resource/common/prop_gift_list/prop_gift_config.json
## 主播特有礼物映射
http://open.douyucdn.cn/api/RoomApi/room/ + roomID
```
